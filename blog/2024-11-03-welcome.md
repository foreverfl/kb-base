## What is Docusaurus?

Docusaurus is a **static-site generator** that helps you build optimized websites quickly. It's perfect for:

- Documentation sites
- Product pages
- Blogs
- Marketing pages
- Any content-centric website

## Features

- **Powered by React** - Extend and customize with React
- **Built with MDX** - Write interactive documents using JSX in Markdown
- **Search** - Your full site is searchable
- **Document Versioning** - Support users on all versions
- **Internationalization** - Translate your site in multiple locales
